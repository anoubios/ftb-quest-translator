# FTB Quest Translator

FTB Quest Translator is a powerful Minecraft mod that allows you to automatically translate quests from the FTB Quests mod directly in-game. This mod is designed to make questing modpacks more accessible to players from all over the world, supporting multiple languages and providing a user-friendly interface for managing translations.

## 🌟 Key Features

- **Instant In-Game Translation**: Translate quest titles, subtitles, and descriptions on the fly while playing.
- **Text-to-Speech (TTS) Support**: The mod can read quest texts aloud in your chosen language. The UI includes visual status indicators (reading, loading, error).
- **Batch Translation**: Translate entire quest chapters at once with a convenient interface for managing the translation cache.
- **Glossary System**: A customizable, language-aware glossary that allows you to preserve original item names or specific terms during translation.
- **Cross-Platform Compatibility**: Supports all major mod loaders and current game versions.

## 📦 Supported Versions

- **Minecraft**: 1.20.1, 1.20.4, 1.21.1
- **Loaders**: Forge, NeoForge, Fabric

## 🔗 Dependencies

To use this mod, the following dependencies must be installed alongside it:
- **FTB Quests**
- **FTB Library**

## 🚀 Installation

1. Ensure you have **FTB Quests** and the appropriate mod loader (Forge, NeoForge, or Fabric) installed for your game version.
2. Download the `FTB Quest Translator` mod file for your specific version and loader.
3. Place the downloaded `.jar` file into your game's `mods` folder.
4. Launch the game.

## ⚙️ Usage

- **Translate a Quest**: Open any quest in the FTB Quests UI. Additional buttons will appear in the quest panel to translate the text.
- **Text-to-Speech (TTS)**: Click the speaker icon next to the quest text to hear it read aloud.
- **Glossary Management**: Add specific terms to the glossary to prevent them from being mistranslated or distorted by machine translation.
- **Batch Translation**: In the mod settings or via the dedicated menu, you can start translating entire chapters and manage your cache (delete unnecessary translations).

## 🛠️ For Developers
